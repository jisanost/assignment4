getElementById, getElementsByClassName, querySelector, and querySelectorAll are JavaScript methods used to select HTML elements from the DOM. getElementById selects a single element using its unique ID. getElementsByClassName selects all elements that have a specific class name. querySelector selects the first element that matches a CSS selector, while querySelectorAll selects all matching elements using CSS selectors.
A new element can be created in JavaScript and inserted into the DOM to display it on a webpage. First, the element is created, then content or attributes can be added to it, and finally it is inserted into a parent element in the document structure.
Event Bubbling is a JavaScript event mechanism where an event starts from the target element and then moves upward through its parent elements step by step. For example, if a button inside a div is clicked, the event first occurs on the button and then bubbles up to the div and other parent elements.
Event Delegation is a technique in JavaScript where a parent element handles events for its child elements instead of adding separate event listeners to each child. It is useful because it improves performance, reduces memory usage, and also works for dynamically added elements.
preventDefault() and stopPropagation() are both event methods in JavaScript, but they work differently. preventDefault() stops the browser’s default action, such as preventing a form submission or link navigation, while stopPropagation() stops the event from moving upward to parent elements during event bubbling.
